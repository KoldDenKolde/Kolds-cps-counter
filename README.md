# Kold's CPS Counter

Fabric client mod for Minecraft 1.21.6 through 1.21.11.

## Features

- CPS counter HUD.
- Tiny old-hitter module from the working compact build.
- Jump reset on `Z`.
- Hitter toggle on `F`.
- Runner/sprint behavior locked on.
- Kold's CPS Counter branding and icon.

## Download

Use the jar in `build/libs/`:

```text
kolds-cps-counter-v1.1.0-1.21.6-1.21.11.jar
```

## Install

Put the jar into your Fabric instance `mods` folder.

Required:

- Minecraft 1.21.6-1.21.11
- Fabric Loader
- Fabric API
- Cloth Config

Optional:

- Mod Menu
